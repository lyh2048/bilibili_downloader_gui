# 哔哩哔哩下载器

🤡：哔哩哔哩视频下载

## 使用

1、安装依赖

```bash
pip install -r requirements.txt
```

`注意：需要用到FFmpeg来合并音频和视频`

👉[FFmpeg](https://ffmpeg.org/)👈

2、运行

```bash
python main.py
```
