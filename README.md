## Installation

1、克隆仓库

```bash
git clone https://github.com/lyh2048/bilibili_downloader_gui.git
```

2、进入目录

```bash
cd bilibili_downloader_gui
```

3、安装依赖

```bash
pip install -r ./requirements.txt
```

Then enjoy it ~

## Usage

```bash
python ./main.py
```
